# Simplex Lite

Commercial-style modular Blender addon package.

## Package Name

`blackwall_simplex_lite`

## Purpose

This build uses a unique internal package name to avoid conflicts with older test installs of `simplex_lite` in Blender.

## Installation

1. Open Blender
2. Go to **Edit > Preferences > Add-ons**
3. Choose **Install from Disk**
4. Select the installable ZIP
5. Enable **Simplex Lite**

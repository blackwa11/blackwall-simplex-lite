# Simplex Lite

Lightweight Blender addon for real-time 4D simplex projection and rotation.

## Installation

Download [blackwall_simplex_lite_installable.zip](https://github.com/blackwa11/blackwall-simplex-lite/releases/download/v1.0.0/blackwall_simplex_lite_installable.zip)

Blender:
Edit → Preferences → Add-ons → Install from Disk

## Features

- Real-time 4D simplex projection
- Mesh / Curve modes
- Six 4D rotation planes
- Lightweight UI

## Package

Modular Blender addon package for experimental higher-dimensional geometry workflows.
